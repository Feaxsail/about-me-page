# Моя персональная страница-визитка

## Описание проекта
Это моя персональная страница-визитка, созданная в рамках самостоятельного задания по веб-разработке. Страница содержит информацию обо мне, моих навыках и контактах.

## Структура проекта
my-webpage/
├── index.html # Главная страница
├── style.css # Стили страницы
├── README.md # Этот файл
└── img/ # Папка для изображений

## Используемые технологии
- HTML5
- CSS3
- Font Awesome для иконок
- Flexbox и Grid для верстки
- Адаптивный дизайн

## Функционал
1. **Шапка профиля**:
   - ФИО и группа
   - Заглушка для фото (иконка пользователя)

2. **Раздел "Обо мне"**:
   - 4 предложения о себе
   - Приятное оформление с карточками

3. **Раздел "Навыки"**:
   - 6 карточек с навыками
   - Анимация при наведении
   - Иконки для каждого навыка

4. **Раздел "Контакты"**:
   - Email (не настоящий)
   - Телефон (не настоящий)
   - GitHub и Telegram
   - Иконки и hover-эффекты

## Особенности
- Адаптивный дизайн (работает на всех устройствах)
- Современный градиентный фон
- Плавные анимации и переходы
- Семантическая верстка
- Чистый и читаемый код

## Как запустить
1. Скачайте файлы проекта
2. Откройте файл `index.html` в любом браузере
3. Или перейдите по ссылке GitHub Pages (если размещено)
